```php yii log/import [filePath] [...options...]```

`[filePath]` (обязательный параметр): Путь к файлу журнала (log file), который вы хотите импортировать в базу данных.

`--log-errors` (необязательная опция, значение по умолчанию: 'no'): Позволяет включить или отключить запись ошибок в журнал. Допустимые значения: "yes" или "no".

`--show-errors` (необязательная опция, значение по умолчанию: 'no'): Позволяет включить или отключить отображение ошибок в консоли. Допустимые значения: "yes" или "no".
